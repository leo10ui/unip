HTML - HyperText Markup Language

O que é HTML?

HTML é um acrônimo para HyperText Markup Language (ou em português Linguagem de Marcação de Hipertexto).
O HTML é muito utilizado para desenvolver páginas web e documentos eletrônicos, como emails, por exemplo.
Seu objetivo é definir a estrutura e formatação, posicionamento dos elementos, e criar blocos e divisões para colocar elementos visuais da página, como imagens, textos, mapas, players, etc.

Histõria do HTML

O HTML foi criado em 1991, por Tim Berners-Lee, no CERN (European Council for Nuclear Research) na
Suíça. Inicialmente o HTML foi projetado para interligar instituições de pesquisa prõximas, e
compartilhar documentos com facilidade. Em 1992, foi liberada a biblioteca de desenvolvimento WWW
( World Wide Web), uma rede de alcance mundial, que junto com o HTML proporcionou o uso em
escala mundial da WEB.

Principais Tags

Tags Estruturais Básicas

  • <!DOCTYPE>: Define o tipo e a versão do documento.
  • <html>: Representa a raiz do documento html.
  • <head>: Contem informação/metadata do documento.
  • <body>: Define o corpo do documento.
  • <div>: Define uma seção do documento.
  • <footer>: Define um rodapé para um documento ou seção.

Exemplo Estrutura Básica

<!DOCTYPE html>
            <html>
                <head>
                    <title>Título da página</title>
                </head>
                <body>
                ..... Conteúdo da página....
                </body>
                <footer>
                </footer>
            </html>

Formatação de Texto

  • <h1> até <h6>: Define o título de uma seção de texto. Com ordem decrescente de importância, usando h1 para títulos principais, e h2 para subtítulos, por exemplo.
  • <p>: Define um parágrafo.
  • <b>: Define uma quebra de linha.
  • <b>: Define texto em negrito.
  • <i>: Define texto em itálico.
  • <p>: Define texto sublinhado.
  • <a>: Define um hyperlink.
  • <pre>: Define texto pré formatado.
  • <code>: Define um pedaço para formatação de código.

Tabelas

  • <table>: Define uma tabela.
  • <td>: Define o cabeçalho de uma tabela.
  • <tr>: Define uma linha de uma tabela.
  • <td>: Define uma célula de uma tabela.

Exemplo de Tabela

<table border="1">
            <tr>
                <th colspan="3">Cabeçalho</th>
            </tr>
            <tr>
                <td>Célula 1</td>
                <td>Célula 2</td>
                <td>Célula 3</td>
            </tr>
        </table>

Elementos Visuais e de Mídia

  • <img>: Define uma imagem.
  • <picture>: Define um container para múltiplas imagens.
  • <embed>: Define um container para uma aplicação externa.
  • <iframe>: Define um quadro embutido.
  • <button>: Define um botão clicável.
